# StructureHelper

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![.NET](https://img.shields.io/badge/.NET-6.0-blue.svg)](https://dotnet.microsoft.com/)
[![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)](https://www.microsoft.com/windows)

**StructureHelper** — это мощное приложение для расчёта и анализа железобетонных сечений, разработанное на WPF (.NET 6). Программа позволяет инженерам-конструкторам моделировать сложные сечения, задавать материалы, нагрузки и выполнять расчёты по предельным состояниям.

## 🌟 Основные возможности

- **Моделирование сечений**: Создание сложных поперечных сечений из различных геометрических примитивов (прямоугольники, круги, стержни арматуры, точки)
- **Библиотека материалов**: Поддержка различных типов материалов:
  - Бетон (Concrete)
  - Арматура (Reinforcement)
  - Упругие материалы (Elastic)
  - Углеволокно (Carbon Fiber)
  - Стекловолокно (Glass Fiber)
- **Шаблоны КЖ**: Быстрое создание типовых железобетонных элементов:
  - Прямоугольные и круглые колонны
  - Балки
  - Плиты перекрытий
- **Расчётные инструменты**:
  - Калькулятор усилий (Force Calculator)
  - Диаграммы взаимодействия (Interaction Diagrams)
  - Расчёт по предельным состояниям
- **Визуализация**: Графическое отображение сечений, эпюр напряжений и результатов расчётов
- **Мультиязычность**: Полная поддержка русского и английского языков интерфейса
- **Гибкие настройки**: Настройка единиц измерения, точности расчётов, визуальных параметров

## 📋 Требования к системе

- **ОС**: Windows 7 или выше
- **.NET**: .NET 6.0 Runtime или выше
- **Память**: Минимум 2 ГБ ОЗУ
- **Дисковое пространство**: 100 МБ свободного места

## 🚀 Установка и запуск

### Вариант 1: Запуск из исходного кода

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd StructureHelper
```

2. Откройте решение `StructureHelper.sln` в Visual Studio 2022 или выше

3. Восстановите NuGet пакеты:
```bash
dotnet restore
```

4. Соберите и запустите проект:
```bash
dotnet run --project StructureHelper/StructureHelper.csproj
```

### Вариант 2: Использование готового исполняемого файла

Если у вас есть скомпилированная версия:

1. Скопируйте папку с приложением на локальный диск
2. Запустите `StructureHelper.exe`
3. При необходимости установите .NET 6.0 Runtime с [официального сайта Microsoft](https://dotnet.microsoft.com/download/dotnet/6.0)

## 📖 Руководство пользователя

Подробное руководство пользователя доступно в папке `Documentation/Manuals/`:
- `UserManual.pdf` — полная документация на русском языке
- `UserManual.docx` — редактируемая версия руководства

### Быстрый старт

1. **Создание нового сечения**:
   - Используйте меню "Элементы" → "Добавить" для добавления геометрических примитивов
   - Или выберите готовый шаблон в меню "Шаблоны"

2. **Назначение материалов**:
   - Откройте панель "Материалы"
   - Добавьте необходимые материалы и назначьте их элементам

3. **Загрузка усилий**:
   - Перейдите в раздел "Расчет"
   - Добавьте калькулятор усилий
   - Задайте значения моментов Mx, My и силы Nz

4. **Выполнение расчёта**:
   - Нажмите кнопку "Запустить расчеты"
   - Просмотрите результаты в табличном и графическом виде

## 🌐 Локализация

Приложение поддерживает два языка интерфейса:
- 🇷🇺 Русский (ru-RU)
- 🇬🇧 Английский (en-US)

Переключение языка осуществляется через кнопку на панели инструментов. Все тексты интерфейса вынесены в файлы ресурсов:
- `Strings.ru-RU.xaml` — русская локализация
- `Strings.en-US.xaml` — английская локализация
